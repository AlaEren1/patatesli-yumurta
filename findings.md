# Findings 🔬

*This file tracks research, API limits, and platform logic.*

- **Supabase Integration**: Required for Authentication and Postgres DB. Use `@supabase/supabase-js`.
- **LLM API Logic**: Open/Gemini API will generate short stories tailored to user parameters (language, topic, difficulty).
- **YouTube API**: Video Syncing will require integrating with the YouTube iframe API and fetching/parsing video transcripts.
- **Reading Module**: Content needs to be parsed into selectable word-nodes to support the Tap-to-Translate functionality.
